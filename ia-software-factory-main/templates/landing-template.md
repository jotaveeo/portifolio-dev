# Landing Page Template

Template para criação de landing pages de alta conversão.

## Stack padrão

| Camada | Tecnologia |
|--------|-----------|
| Framework | Next.js |
| Estilização | Tailwind CSS |
| Animações | Framer Motion |
| Ícones | Lucide Icons |
| Deploy | Vercel |

## Seções obrigatórias

1. **Hero** — Headline, sub-headline, CTA principal
2. **Social Proof** — Logos de clientes ou métricas
3. **Features** — 3-6 funcionalidades principais
4. **How it works** — Passos do fluxo
5. **Testimonials** — Depoimentos de usuários
6. **Pricing** — Planos e preços
7. **FAQ** — Perguntas frequentes
8. **CTA Final** — Chamada para ação final
9. **Footer** — Links, social, legal

## Boas práticas

- Headline clara e orientada a benefício
- CTA visível e com contraste
- Prova social logo no início
- Mobile-first design
- Carregamento rápido (< 3s)
- SEO otimizado (meta tags, OG, structured data)
