# Stack Defaults

Stack tecnológica padrão para sistemas gerados pela AI Software Factory.

## Frontend

| Tecnologia | Uso |
|------------|-----|
| React / Next.js | Framework principal |
| Tailwind CSS | Estilização |
| TypeScript | Linguagem |
| Zustand / Redux | Estado global |
| React Query | Fetch e cache |
| Zod | Validação |

## Backend

| Tecnologia | Uso |
|------------|-----|
| Node.js | Runtime |
| Fastify / Express | Framework HTTP |
| TypeScript | Linguagem |
| Prisma / TypeORM | ORM |
| Zod | Validação de entrada |
| Jest | Testes |

## Banco de dados

| Tecnologia | Uso |
|------------|-----|
| PostgreSQL | Banco relacional principal |
| Redis | Cache e sessões |
| MongoDB | Quando NoSQL for mais adequado |

## Infraestrutura

| Tecnologia | Uso |
|------------|-----|
| Docker | Containerização |
| Docker Compose | Orquestração local |
| Nginx | Reverse proxy |
| GitHub Actions | CI/CD |
| Vercel / Railway | Deploy |

## Autenticação

| Tecnologia | Uso |
|------------|-----|
| JWT | Tokens de acesso |
| bcrypt | Hash de senhas |
| OAuth2 | Login social |

## Observabilidade

| Tecnologia | Uso |
|------------|-----|
| Winston / Pino | Logging |
| Sentry | Error tracking |
| Prometheus + Grafana | Métricas |
