# Engineering Principles

Todos os agentes devem seguir estes princípios:

- Clean Architecture
- SOLID principles
- Separation of concerns
- Modular code structure
- Testability
- Maintainability
- Scalability
- Security by design

Todo código gerado deve ser:

- Production-ready
- Bem documentado
- Modular
- Seguir boas práticas modernas
