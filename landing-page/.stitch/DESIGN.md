# Design System Notes for Stitch Generation

## OVERALL VIBE
Modern, clean, trustworthy, and tech-forward. Inspired by Google's DeepMind Antigravity aesthetics. 
It should feel like an elite engineering tool blended with a high-end educational platform.

## CORE THEME
- Platform: Web, Desktop-first focusing on responsiveness.
- Theme: Light mode primarily, with very stark high-contrast components for technical sections.

## TYPOGRAPHY
- Headings: Clean sans-serif (resembling Google Sans Flex).
- Body: Accessible, spacious sans-serif (Google Sans).
- Headings are large, confident (e.g. H1 at 80px).

## COLOR PALETTE
- **Background**: Soft Gray (`#EDEDED`)
- **Primary Text**: Heavy Charcoal/Graphite (`#121317`)
- **Accent/Link**: Vivid Light Blue (`#2FA1D6`)
- **Brand Secondary**: Soft Sand (`#C5BDAD`)

## COMPONENT STYLING
- **Buttons (Primary Action)**: Pill-shaped (`border-radius: 9999px`), very subtle, almost flat. Black (`#1F1F1F`) background with stark white text. No extreme shadows.
- **Buttons (Secondary)**: Off-white (`#F8F9FC`) background with black text (`#121317`). Pill-shaped.
- **Cards & Containers**: Subtle separation from background, slightly rounded corners (`4px` or `8px`), maintaining a brutalist meets modern-minimalist approach.

## LAYOUT
Spacing relies heavily on a 4px base-unit system. generous padding between distinct sections to allow the content and typography to breathe.
